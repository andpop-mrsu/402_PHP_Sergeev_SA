# Task03

Лабораторная работа №3 по технологиям разработки серверных приложений на PHP.

## Описание

Проект представляет собой одностраничное веб-приложение (SPA) с REST API для игры.  
Серверная часть реализована на PHP с использованием Slim, данные хранятся в SQLite.

## Стек технологий

- PHP 8+
- Slim 4
- SQLite
- JavaScript
- HTML/CSS

## Структура проекта

- `public/` — публичная часть приложения
- `src/` — серверная логика
- `db/` — база данных SQLite
- `vendor/` — зависимости Composer

## Установка зависимостей

Перед запуском необходимо установить зависимости через Composer:

```bash
composer install
```

## Запуск проекта

Из корневой папки проекта выполните:

```bash
php -S localhost:3000 -t public
```

После этого приложение будет доступно по адресу:

```text
http://localhost:3000
```

## Возможные проблемы

Для работы SQLite в PHP должны быть включены расширения:

- `pdo_sqlite`
- `sqlite3`

Проверить это можно командой:

```bash
php -m
```

## REST API

Приложение поддерживает следующие маршруты:

- `GET /games` — получить список игр
- `GET /games/{id}` — получить информацию об игре
- `POST /games` — создать новую игру
- `POST /step/{id}` — выполнить ход в игре

## Примечание

Папка `vendor` и файл базы данных не добавляются в репозиторий, так как исключены через `.gitignore`.
